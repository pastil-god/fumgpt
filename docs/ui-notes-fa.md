# یادداشت بازطراحی UI برای FumGPT

در این نسخه ظاهر فروشگاه از حالت اسکلت ساده خارج شده و به یک لندینگ/استورفرانت حرفه‌ای‌تر تبدیل شده است:

- هدر استیکی با CTA مشخص
- هیرو قوی با ارزش پیشنهادی روشن
- کارت‌های دسته‌بندی و شلف محصولات
- قیمت قبل/بعد و تخفیف روی هر محصول
- رزرو مسیر توسعه برای آکادمی و بازارچه ایجنت
- برندینگ جدید FumGPT با لوگوی اختصاصی
