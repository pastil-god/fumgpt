# FumGPT Phase 1 Storefront

نسخه بازطراحی‌شده فاز اول فروشگاه روی Next.js.

## ویژگی‌های فعلی

- برند جدید FumGPT
- UI/UX بازطراحی‌شده با ظاهر حرفه‌ای‌تر
- وارد شدن 7 محصول واقعی از سایت فعلی شما
- صفحات خانه، محصولات، جزئیات محصول، ورود، سبد خرید، آکادمی و بازارچه ایجنت
- جایگاه آماده برای اتصال درگاه و پیامک در فاز بعدی

## اجرا

```bash
pnpm install
pnpm dev
```

سپس در مرورگر باز کنید:

```bash
http://localhost:3000
```
